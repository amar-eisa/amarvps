## تطوير النظام من عرض إحصائيات إلى أداة تحكم فعّالة

### الرؤية

تحويل اللوحة من **"قارئ للبيانات"** إلى **"مركز تحكم وتنبيه ذكي"** يتفاعل مع السيرفر ويحميه.

---

### المجموعة 1: التنبيهات والمراقبة الذكية (الأولوية القصوى)

**2. سجل الأحداث (Event Log)**

- حفظ كل حدث مهم في قاعدة البيانات (حاوية توقفت، ذاكرة تجاوزت الحد، دخول مستخدم جديد)
- صفحة تاريخ الأحداث مع فلترة بالتاريخ والنوع والخطورة

---

### المجموعة 2: التحكم عن بُعد (Control)

**4. إدارة الحاويات (Docker Actions)**

- أزرار: Start / Stop / Restart / Remove لكل حاوية
- عرض logs الحاوية مباشرة في واجهة منبثقة

**5. إدارة الخدمات (systemctl)**

- restart لخدمات systemd من الواجهة
- عرض حالة الخدمات (active/failed/inactive)

---

### المجموعة 3: تحليلات تاريخية (Analytics)

**8. تقارير دورية**

- تقرير أسبوعي PDF/Email: متوسط الاستخدام، أعلى القمم، الحاويات الأكثر استهلاكاً
- مقارنة بين فترات زمنية

**9. ترتيب الحاويات حسب الاستهلاك**

- top CPU consumers
- top RAM consumers
- top network consumers

### المجموعة 6: تحسينات UX

**15. بحث عام**

- بحث سريع في الحاويات / البورتات / الأوامر / المستخدمين

**16. تصدير البيانات**

- تصدير الجداول إلى CSV / JSON

**17. Dark/Light Mode محسّن + شاشة كاملة**

- تثبيت اللوحة كتطبيق على الموبايل مع push notifications
